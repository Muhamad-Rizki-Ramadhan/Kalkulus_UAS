Konsep Dasar Turunan (Derivatives)

1. Pengertian Turunan

Apa itu turunan?
Sederhananya, turunan adalah ukuran yang menunjukkan seberapa cepat suatu nilai berubah seiring dengan perubahan nilai lainnya (tingkat perubahan sesaat).

Apa fungsinya di matematika?
Turunan digunakan untuk menemukan gradien (kemiringan) garis singgung pada suatu kurva di titik tertentu, serta untuk menganalisis bentuk grafik (apakah grafiknya naik atau turun).

Apa kegunaan nyata dari turunan?
Dalam kehidupan nyata, turunan bisa digunakan untuk menghitung kecepatan kendaraan pada detik tertentu secara akurat. Dalam bidang teknologi, konsep ini berguna untuk proses optimasi (misalnya mencari efisiensi waktu eksekusi maksimal atau penggunaan memori minimal).

2. Rumus Dasar Turunan

Definisi Bentuk Umum (Limit)
Secara formal, turunan dari fungsi f(x) dilambangkan dengan f'(x) dan didefinisikan menggunakan limit selisih kuosen:

f'(x) = limh → 0
f(x + h) - f(x) h

Rumus Dasar (Aturan Pangkat)
Untuk perhitungan cepat pada fungsi aljabar berbentuk pangkat seperti f(x) = axn, rumus cepat turunannya adalah dengan mengalikan pangkat ke depan lalu pangkatnya dikurangi satu:

f'(x) = a · n · xn-1

3. Aturan Rantai (Fungsi Komposisi/Berpangkat)

Jika sebuah fungsi memiliki pangkat dan di dalamnya terdapat fungsi lagi (bukan cuma x biasa), kita harus menggunakan Aturan Rantai. Untuk caranya adalah dengan cara turunkan pangkatnya dulu, lalu kalikan dengan turunan fungsi di dalamnya.

f(x) = [u(x)]n
f'(x) = n · [u(x)]n-1 · u'(x)

4. Turunan Fungsi Perkalian

Untuk fungsi yang berbentuk perkalian (dua fungsi dikalikan), kita tidak bisa menurunkan kedua fungsi secara terpisah lalu mengalikan hasilnya. Misalkan fungsi pertama adalah u dan fungsi kedua adalah v, kita harus gunakan rumus perkalian (product rule) ini:

f(x) = u(x) · v(x) = u · v
f'(x) = u' · v + u · v'

5. Turunan Fungsi Pecahan (Pembagian)

Untuk fungsi yang berbentuk pecahan (pembilang dan penyebut sama-sama memiliki variabel x), kita tidak bisa menurunkan atas dan bawahnya secara terpisah. Misalkan bagian atas adalah u dan bawah adalah v, kita harus gunakan rumus ini:

f(x) =
u(x) v(x)
=
u v

f'(x) =
(u' · v) - (u · v') v2

6. Turunan Trigonometri

Fungsi trigonometri memiliki pola turunan dasar yang wajib diingat. Berikut adalah keenam turunan dasarnya:

  • Turunan dari sin x adalah cos x
  • Turunan dari cos x adalah -sin x
  • Turunan dari tan x adalah sec2 x
  • Turunan dari sec x (secan) adalah sec x · tan x
  • Turunan dari csc x (cosecan) adalah -csc x · cot x
  • Turunan dari cot x (cotangen) adalah -csc2 x
    •

Tips menghafal : Setiap turunan fungsi trigonometri yang berawalan huruf "c" (cos, csc, cot), hasilnya pasti negatif.

Aplikasi Turunan

Rumus-rumus penting dalam aplikasi turunan, mulai dari persamaan garis singgung hingga nilai stasioner:

A. Garis Singgung & Garis Normal

  • Gradien Garis Singgung : m = f'(x)
  • Persamaan Garis Singgung : y - y1 = m (x - x1)
  • Gradien Garis Normal : m1 = -1/m2 = -1/f'(x)
  • Persamaan Garis Normal: y - y1 = mn(x - x1)

B. Gradien & Hubungan Dua Garis

  • Gradien dari y = ax + b adalah m = a
  • Gradien dari ax + by + c = 0 adalah m = -a/b
  • Dua garis saling sejajar: m1 = m2
  • Dua garis tegak lurus: m1 = -1/m2

C. Kurva Naik, Turun & Stasioner

  • Syarat Kurva Naik: f'(x) > 0
  • Syarat Kurva Turun: f'(x) < 0
  • Syarat Titik Stasioner: f'(xᵢ) = 0
  • Titik Balik Maksimum: f''(x) < 0
  • Titik Belok Horizontal: f''(xᵢ) = 0
  • Titik Balik Minimum: f''(xᵢ) > 0

Tips: Pastikan kamu menguasai turunan dasar terlebih dahulu sebelum masuk ke materi ini, karena semuanya sangat bergantung pada hasil pencarian turunan pertama f`(x) dan turunan kedua f``(x).

Panduan Penulisan Fungsi
  • Gunakan ^ untuk pangkat. Contoh: x kuadrat ditulis x^2.
  • Gunakan * untuk perkalian variabel. Contoh: 2x² + 2x + 2 ditulis 2*x^2 + 2*x + 2. Berlaku untuk trigonometri juga 2 sin(x) jadi 2*sin(x).
  • Untuk trigonometri, wajib gunakan tanda kurung. Contoh: sin(x), cos(x).

Contoh Soal

Soal 1

Tentukan turunan pertama dari fungsi f(x) = 3x3 + 5x2 - 4x + 10!

Lihat Langkah Penyelesaian

Gunakan aturan perkalian pangkat n · xn-1 untuk setiap suku:

  • Turunan dari 3x3 adalah 3 · 3x2 = 9x2
  • Turunan dari 5x2 adalah 5 · 2x = 10x
  • Turunan dari -4x adalah -4
  • Turunan dari konstanta 10 adalah 0

Jadi, f'(x) = 9x2 + 10x - 4

Soal 2 (Aturan Rantai)

Tentukan turunan pertama dari f(x) = (2x + 3)4

Lihat Langkah Penyelesaian

Misalkan u(x) = 2x + 3, maka turunan dalamnya adalah u'(x) = 2. Terapkan aturan rantai:

  • Turunkan pangkat luar: 4(2x + 3)3
  • Kalikan dengan turunan dalam (u'): 4(2x + 3)3 · 2
  • Kalikan angka depan: 4 · 2 = 8

Jadi, f'(x) = 8(2x + 3)3

Soal 3 (Pecahan)

Tentukan turunan dari f(x) = (3x) / (x - 2)

Lihat Langkah Penyelesaian

Tentukan u dan v beserta turunannya:

  • u = 3x → u' = 3
  • v = x - 2 → v' = 1

Masukkan ke rumus pembagian:

f'(x) =
3(x - 2) - 3x(1) (x - 2)2


f'(x) =
3x - 6 - 3x (x - 2)2

Jadi, f'(x) = -6 (x - 2)2

Soal 4 (Trigonometri)

Tentukan turunan pertama dari f(x) = 5 sin x + 2 cos x

Lihat Langkah Penyelesaian

Turunkan masing-masing suku sesuai aturan dasar trigonometri:

  • Turunan dari 5 sin x adalah 5 cos x
  • Turunan dari 2 cos x adalah 2(-sin x) = -2 sin x

Jadi, f'(x) = 5 cos x - 2 sin x

Soal 5 (Perkalian)

Tentukan turunan pertama dari f(x) = (2x + 1)(x2 + 3)

Lihat Langkah Penyelesaian

Tentukan u dan v beserta turunannya menggunakan aturan perkalian f'(x) = u'v + uv':

  • u = 2x + 1 → u' = 2
  • v = x2 + 3 → v' = 2x

Masukkan ke rumus perkalian:

f'(x) = u'v + uv'

f'(x) = 2(x2 + 3) + (2x + 1)(2x)

f'(x) = 2x2 + 6 + 4x2 + 2x

f'(x) = 6x2 + 2x + 6

Jadi, f'(x) = 6x2 + 2x + 6
